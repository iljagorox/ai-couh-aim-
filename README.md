🛠️ Фундамент (Обязательно к установке)
Это база, без которой проект не запустится.

1. Python 3.10+

Зачем: Основной язык, на котором написан проект. Рекомендую версии 3.10-3.12 как самые стабильные.

🔗 Ссылка: https://www.python.org/downloads/

💡 Совет: При установке обязательно поставьте галочку "Add Python to PATH" — это сэкономит вам много времени и нервов в будущем.

2. Ollama (основной фреймворк для запуска ИИ)

Зачем: Это сервер, который запускает и обслуживает все языковые модели (LLM) прямо на вашем компьютере.

🔗 Ссылка: https://ollama.com/download

💡 Совет: Просто скачайте установщик OllamaSetup.exe и запустите его. После установки в системном трее появится иконка Ollama — значит, сервер работает в фоне.

3. Модель ИИ: gemma3:4b-it-qat

Зачем: Это и есть "мозг" вашего ИИ-агента. Модель понимает текст и изображения, при этом достаточно лёгкая для запуска на обычных компьютерах.

🔗 Команда для загрузки: ollama pull gemma3:4b-it-qat

💡 Совет: Эту команду нужно выполнить в командной строке (CMD) или PowerShell после установки Ollama. Загрузка может занять время, так как модель весит несколько гигабайт.

🔌 Основные Python-библиотеки
Следующий шаг — установка Python-библиотек, указанных в requirements.txt. Это делается одной командой в терминале (подробнее в конце раздела), но важно понимать, для чего нужна каждая из них.

customtkinter: Создаёт современный графический интерфейс приложения.

psutil: Позволяет ИИ-агенту "видеть" запущенные процессы (игры, браузеры).

pyautogui: Управляет мышью и клавиатурой, эмулируя действия пользователя.

ollama: Официальная библиотека для общения Python-кода с сервером Ollama.

Pillow: Инструмент для обработки скриншотов перед отправкой модели.

pywin32 (>=306): Обеспечивает доступ к API Windows (нужно для получения информации об активном окне).

pygetwindow: Упрощает работу с окнами в Windows (переключение, фокусировка).

🎤 Компоненты для голосового переводчика
Если планируете использовать функцию перевода речи, потребуются дополнительные библиотеки.

faster-whisper: Высокопроизводительная версия Whisper для распознавания речи.

sounddevice: Захватывает звук с микрофона.

silero-vad: Умный детектор голоса, который "отрезает" тишину.

torch: Фреймворк машинного обучения (фундамент для faster-whisper). Для обычного ПК подойдёт CPU-версия.

soundfile: Сохраняет и читает аудиофайлы различных форматов.

onnxruntime: Среда выполнения для оптимизированных нейросетей (используется VAD-моделью).

💡 Как установить всё и сразу: Откройте терминал в папке с проектом и выполните:

bash
pip install -r requirements.txt
Эта команда сама установит все основные библиотеки. Для голосового переводчика, скорее всего, потребуется вручную доустановить faster-whisper и silero-vad, так как они не всегда включаются в базовый список зависимостей.

✨ Советы для стабильной работы
Используйте виртуальное окружение Python (venv). Это изолирует проект от остальной системы и предотвратит конфликты библиотек.

Установите Microsoft Visual C++ Redistributable. Многие библиотеки (особенно связанные с ИИ) требуют этот пакет для работы на Windows. Скачать можно с официального сайта Microsoft.
